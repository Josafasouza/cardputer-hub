M5 Cardputer: O Universo Maker no seu Bolso.

Uma plataforma de desenvolvimento portátil e poderosa baseada no ESP32-S3. De automação a jogos retro, seus projetos começam aqui. Explore, crie e inove.

Comece a Programar Ver Projetos

Especificações Técnicas ⚙️

CPU

ESP32-S3, Dual-Core @ 240MHz

Memória

8MB PSRAM, 16MB Flash

Tela

1.14" TFT LCD (240x135)

Teclado

QWERTY com 56 teclas

Conectividade

Wi-Fi & Bluetooth 5

Portas

USB-C, MicroSD, Grove

Extras

Microfone, IR, Alto-falante

Downloads e Ferramentas 🚀

Software de Gravação (Burner)

M5Burner v3

A ferramenta oficial da M5Stack para gravar o firmware de forma rápida e fácil. Indispensável para começar.

Download M5Burner

Ambientes de Desenvolvimento (IDEs)

Arduino IDE

O ambiente clássico e versátil para programação em C/C++. Perfeito para acessar a vasta biblioteca de projetos da comunidade Arduino.

URL para o Gerenciador de Placas: https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/arduino/package_m5stack_index.json

Download Arduino IDE

UIFlow 2.0

A plataforma de programação visual (blockly) e MicroPython da M5Stack. Ideal para iniciantes e prototipagem rápida.

Acessar UIFlow 2.0 (Web)

PlatformIO

Uma alternativa poderosa ao Arduino IDE, integrada ao VS Code. Oferece depuração avançada e gerenciamento de bibliotecas.

Instalar PlatformIO
